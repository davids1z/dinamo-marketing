### SWOT Analiza

**Snage:**
- Najuspješniji hrvatski klub svih vremena
- Jedna od najjačih akademija u Europi (Modrić, Kovačić, Gvardiol, Olmo)
- Redoviti sudionik UCL/EL — daje europsku vidljivost
- Efekt halo-a od uspjeha hrvatske repre (WC 2018 finale, 2022 bronca)
- Već postojeća platforma s 14 modula

**Slabosti:**
- Engagement pada drastično između europskih utakmica
- Mali domaći liga (samo 10 timova, 180 utakmica)
- Hajduk ima veću navijačku bazu unatoč manje uspjeha
- Ograničen budžet u usporedbi s top 32 europska kluba

**Prilike:**
- 3.2M dijaspore — gotovo isto kao populacija Hrvatske
- World Cup 2026 u SAD-u — boost za hrvatski nogomet
- AI automatizacija smanjuje troškove produkcije sadržaja 10x
- Sport marketing tržište raste 7.94% CAGR (do $195.5B 2032)
- Digitalni fan engagement raste 16.87% CAGR
- Nema sličnog alata na tržištu koji sve objedinjuje za sportske klubove

**Prijetnje:**
- Konkurencija od enterprise alata (Hootsuite $99/mo, Sprout $199/seat)
- Sportski specifični alati (Greenfly, WSC Sports) za velike klubove
- API ograničenja socijalnih mreža se stalno mijenjaju
- Algoritam promjene na platformama

### Konkurentska prednost naše platforme

Naša platforma popunjava **prazninu na tržištu** jer:
1. **Nijedan alat ne kombinira sve** — klubovi koriste 3-5 alata (Hootsuite + Canva + Greenfly + Blinkfire + Buffer)
2. **AI-first pristup** — automatska generacija sadržaja, vizuala i videa
3. **Sportski specifičan** — razumije matchday cikluse, content pillare, sezonu
4. **Pristupačan** — dizajniran za klubove srednje veličine, ne samo elitu
5. **End-to-end** — od ideje do objave do analize, sve na jednom mjestu

## 5. AUTOMATSKI SUSTAV - Kako bi sve funkcioniralo

### Arhitektura Auto-Publishing Sustava

```
┌─────────────────────────────────────────────────────────────┐
│                    CONTENT PIPELINE                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  1. AI GENERACIJA PLANA                                     │
│     ├── OpenRouter/Gemini generira mjesečni plan            │
│     ├── Plan se sprema u ContentPlan (DB)                   │
│     └── Svaki post = ContentPost s statusom "draft"         │
│                                                             │
│  2. AI KREACIJA SADRŽAJA                                    │
│     ├── Za svaki ContentPost:                               │
│     │   ├── Generira caption (HR/EN/DE) via AI              │
│     │   ├── Generira visual brief → Image Gen API           │
│     │   │   ├── Stability AI za fotografije                 │
│     │   │   ├── OpenAI DALL-E za grafike                    │
│     │   │   └── Template engine za branded content          │
│     │   ├── Video montaža (FFmpeg + AI):                    │
│     │   │   ├── Auto-crop 9:16 / 16:9 / 1:1                │
│     │   │   ├── Dodaj branded overlay                       │
│     │   │   ├── Dodaj caption/subtitle                      │
│     │   │   └── Compress za platformu                       │
│     │   └── Status → "ready_for_review"                     │
│     │                                                       │
│  3. APPROVAL WORKFLOW                                       │
│     ├── Auto-approve (ako content score > threshold)        │
│     ├── Manual review queue za edgy content                 │
│     └── Status → "approved" ili "revision_needed"           │
│                                                             │
│  4. AUTO-PUBLISH                                            │
│     ├── Celery Beat provjerava svakih 5 min                 │
│     ├── Za approved + scheduled content:                    │
│     │   ├── Meta Graph API → Instagram + Facebook           │
│     │   ├── TikTok Content Posting API → TikTok             │
│     │   ├── YouTube Data API → YouTube/Shorts               │
│     │   └── Buffer API → backup/fallback                    │
│     ├── Status → "published"                                │
│     └── Sprema platform post IDs za tracking                │
│                                                             │
│  5. PERFORMANCE TRACKING (Real-time)                        │
│     ├── Celery task svaki sat vuče metrike:                 │
│     │   ├── Impressions, reach, engagement                  │
│     │   ├── Clicks, CTR, CPC, ROAS                         │
│     │   ├── Video views (25%, 50%, 75%, 100%)               │
│     │   ├── Comments, shares, saves                         │
│     │   └── Follower growth                                 │
│     ├── PostMetric & AdMetric tablice                       │
│     └── Dashboard real-time refresh                         │
│                                                             │
│  6. AI OPTIMIZACIJA                                         │
│     ├── Analizira performance podatke                       │
│     ├── Automatski:                                         │
│     │   ├── Pauzira loše ads (CTR < threshold)              │
│     │   ├── Skalira dobre ads (ROAS > target)               │
│     │   ├── Refresha creative koji je "umoran"              │
│     │   └── Predlaže optimizacije za sljedeći mjesec        │
│     └── Mjesečni AI report s preporukama                    │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### API Integracije za Auto-Publishing

**Meta Graph API (Instagram + Facebook):**
```
POST /{ig-user-id}/media              → Upload slike/videa
POST /{ig-user-id}/media_publish      → Objavi post
POST /{page-id}/feed                  → Facebook post
POST /{page-id}/photos                → Facebook fotografija
POST /{page-id}/videos                → Facebook video
GET  /{media-id}/insights             → Dohvati metrike
```

**TikTok Content Posting API:**
```
POST /v2/post/publish/inbox/video/init  → Inicijaliziraj upload
POST /v2/post/publish/video/init        → Direktna objava
GET  /v2/post/publish/status/fetch      → Status objave
GET  /v2/video/query                    → Dohvati video info
```

**YouTube Data API:**
```
POST /upload/youtube/v3/videos         → Upload video
POST /youtube/v3/thumbnails/set        → Postavi thumbnail
GET  /youtube/v3/videos?part=statistics → Dohvati statistike
```
